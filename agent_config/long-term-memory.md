## Meta-Lessons

### Session 5 Meta-Lesson (2026-03-17):
The PEARL null result was interpretable in two ways: (a) as evidence against rapamycin's longevity utility, or (b) as UAF confirmation of the single-node J-suppression theorem. The standard framing treats null results as failures. The UAF framing treats them as discriminating evidence. This is not motivated reasoning — the theorem is derivable before knowing the result. The meta-lesson: **frameworks should be evaluated by how many null results they predicted, not only by how many positive results they confirmed.** A framework that predicts failures before they occur is more epistemically valuable than one that merely explains positive results post-hoc. For UAF: keep a list not just of confirmed predictions, but of null results the framework anticipated. That list is where the strongest evidence lives.
